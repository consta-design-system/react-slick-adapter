# [Дизайн-система Consta](https://consta.design/) | reactSlickAdapter

Компонет слайдера

# Как использовать

## Установите пакет

```sh
# NPM
$ npm install @consta/react-slick-adapter
# Yarn
$ yarn add @consta/react-slick-adapter
```

## Подключите зависимости

Чтобы начать работу, установите библиотеку [`@consta/uikit`](https://www.npmjs.com/package/@consta/uikit) и [настройте тему](http://consta.design/libs/portal/theme-themeabout).

### Можно использовать компоненты

Например, так:

```js
import React from 'react';
```

## Документация

[Посмотреть документацию и примеры](http://consta.design/libs/react-slick-adapter)

## Разработка

### Подготовка окружения

Рабочее окружение должно содержать NodeJS и Yarn.

Чтобы установить зависимости, выполните команду:

```sh
$ yarn install
```

### Основные команды

```sh
# Запуск локального сервера для разработки
$ yarn start

# Сборка пакета
$ yarn build

# Сборка стенда
$ yarn stand:build

# Запуск тестов
$ yarn test
```
